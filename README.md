### 功能
- 车辆名称
- 车辆状态（在线 / 哨兵 / 休眠 / 进入休眠 / 驾驶中 / 充电中 / 更新中 / 离线）
- 电池状态（剩余电量比例 / 剩余里程数 / 充电上限阈值）
- 充电状态（功率 / 充电上限 / 剩余时间）
- 控制状态（车锁 / 主驾有人 / 车窗 / 空调 / 车门）
- Widget 最后更新时间
- 当前位置地名
- 当前位置地图（高德地图）
- 当前朝向
